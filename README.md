# 🐍 Snake Game – Mobile & Desktop Platform

![Profile Picture](https://www.istockphoto.com/illustrations/default-profile-picture-pictures)

This project is a classic **Snake Game** developed with **Unity 6**.  
The game is designed to run on both **Desktop** and **Mobile** platforms.

The player controls a snake that moves around the screen, eats food, grows longer, and tries to avoid collisions.

---

## 🎮 Features

- Classic Snake gameplay
- Random food spawning
- Snake body growth
- Score system
- Game Over system
- Cross-platform support (Desktop & Mobile)

---

## 🛠️ Technologies Used

- Unity 6
- C#
- Git
- GitHub
- Visual Studio / VS Code
